# BugCord


Realtime chat app with websockets using Node.js


## Technologies used

* Node.js
* socket.io
* express.js
* vanilla.js for Frontend


## Usage:
To view the hosted website click here : https://bugcord.herokuapp.com/
          
          
          
To run app locally :
```
npm install
npm run dev
```
open browser to  localhost:3000
